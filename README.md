# HEASARC general utility scripts

This package includes some general utilities for handling various HEASARC tasks. These scripts are useful for users downloading data, searching the [HEASARC Browse interface](https://heasarc.gsfc.nasa.gov/cgi-bin/W3Browse/w3browse.pl), and using other tools like the [XSpec X-ray spectral modeling software](https://heasarc.gsfc.nasa.gov/docs/software/xspec/index.html) and the [XSelect X-ray data filtering/selection software](https://heasarc.gsfc.nasa.gov/docs/software/ftools/xselect/index.html).

## Contents

- archive/    - scripts to work on the archive
- caldb/      - scripts related to [CALDB](https://heasarc.gsfc.nasa.gov/docs/heasarc/caldb/) files
- xspec/      - utilities related to [Xspec](https://heasarc.gsfc.nasa.gov/docs/software/xspec/index.html)
- other/      - misc other scripts
